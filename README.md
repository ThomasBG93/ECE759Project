# ECE759Project

## Programs:
  * randMatrix.c
    * ./randMatrix.exe r c
    *   input: row and col size
    *   output: random matrix of row x col, output to crtMatrix.txt
  * mvMatrix.c
    * ./mvMatrix r c mov
    *  input: row size, col size, amount of moves to make
    *  output: multiple matricies to the same file movingMatrix.txt
